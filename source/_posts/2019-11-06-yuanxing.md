### 创建对象
我们可以通过new的方式创建一个对象
> var obj = new Object();

也可以通过对象字面量的形式创建一个简单的对象
> var obj = {}

### 工厂模式
工厂模式就是我们提供一个模子，然后通过这个模子复制我们需要的对象。
```javascript
var createPerson = function(name,age) {
// 声明一个中间对象，该对象为工厂模式的模子
var o = new Object();
//依次添加我们需要的属性与方法
o.name = name;
o.age = age;
return o;
}

//创建两个实例
var perTom = createPerson('Tom',20)
var perJake = createPerson('jake',22)

```
